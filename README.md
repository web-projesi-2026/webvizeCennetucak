# Cennet Uçak – Kişisel Portfolyo Web Sitesi

## Proje Açıklaması
Bu proje, Cennet Uçak için hazırlanmış kişisel portfolyo web sitesidir. Site içerisinde kendimi, yeteneklerimi, projelerimi ve iletişim bilgilerimi modern ve kullanıcı dostu bir arayüz ile sunuyorum. Responsive yapıya sahip olan bu web sitesi, hem masaüstü hem mobil cihazlarda düzgün çalışacak şekilde tasarlanmıştır.

## Hedef Kullanıcı
- Cennet Uçak’ın çalışmalarını incelemek isteyen öğretmenler, işverenler ve geliştiriciler
- Projeler hakkında bilgi almak isteyen ziyaretçiler
- Benimle iletişime geçmek isteyen kişiler

## Temel Özellikler
- **Çoklu Sayfa Yapısı:** Ana Sayfa, Hakkımda, Projeler, İletişim
- **Modern Tasarım:** Şık, sade ve dikkat çekici arayüz
- **Responsive Tasarım:** Telefon, tablet ve bilgisayarda uyumlu görünüm
- **Navigasyon Menüsü:** Sayfalar arasında kolay geçiş
- **İletişim Formu:** Ziyaretçilerin mesaj gönderebilmesini sağlar
- **CSS Animasyonları:** Sayfa geçişleri ve içerik görünüm efektleri
- **JavaScript Desteği:** Etkileşimli yapı ve dinamik öğeler
- **Parçacık Arka Plan Efekti:** Görsel açıdan daha modern bir görünüm
- **Kişisel Tanıtım Alanı:** Hakkımda bilgilerinin sunulması
- **Proje Tanıtım Alanı:** Yapılan çalışmaların sergilenmesi

## Kullanılan Teknolojiler
- HTML5
- CSS3
- JavaScript
- FormSubmit (iletişim formu için)
